## Installation

1. Clone this repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   streamlit run app.py
   ```

## Dataset

The application uses the TB_Burden_Country.csv dataset from the World Health Organization, containing tuberculosis burden estimates by country and year.

## Learning Objectives

This application covers:
- **SQL Fundamentals**: Querying and manipulating data
- **Descriptive Statistics**: Understanding data distributions
- **Data Cleaning**: Handling missing values and data preprocessing
- **Python Libraries**: Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
- **Data Visualization**: Creating effective charts and graphs
- **Machine Learning**: Building and evaluating predictive models

## Usage

Navigate through different sections using the sidebar to explore various data science concepts. Each section provides interactive tools to learn and practice fundamental skills.
